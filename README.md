## 🧭 What this app does

takingnotes is a browser-based drawing and animation app for smart notebooks. It is built for the Huion X10 and the Wacom Slate and Spark series.

Use it to sketch, edit layers, and build simple animations from your notes and drawings. It runs in a web browser, so you do not need to install a full desktop app.

## 🖱️ How to use it

After you open the app, use it like this:

1. Connect your smart notebook device to your computer.
2. Open your notes or start a new page.
3. Pick a pen, brush, or tool from the toolbar.
4. Draw on the canvas.
5. Add a new layer if you want to keep parts separate.
6. Move between frames in the animation timeline if you want motion.
7. Save your work often.
8. Export your drawing when you are done.

## 🛠️ Troubleshooting

### App does not open

- Check that the files finished downloading
- Extract the ZIP file first if you downloaded one
- Open the correct folder again
- Try a different browser

### Device is not detected

- Reconnect the notebook device
- Unplug and plug it back in
- Close and reopen the browser
- Try a different USB port
- Make sure the device is powered on

### Drawing feels slow

- Close other browser tabs
- Use a recent browser version
- Reduce other apps running on Windows
- Try a smaller canvas if your work is very large

### Saved file is hard to find

- Check your Downloads folder
- Search for the file name in Windows
- Save exports to Desktop while testing

## 📝 Project background

This project was built from research and testing on real notebook hardware. It uses React, TypeScript, and Vite. Wacom SmartPad support was informed by the tuhi project.

The app focuses on a simple drawing workflow for people who want to sketch, write, and animate in one place.
